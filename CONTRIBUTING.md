# Contributing

Thanks for your interest in contributing to the Marketing Content Repo Template.

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b my-change`)
3. Make your changes
4. Commit with a clear message
5. Open a pull request

## What Makes a Good Contribution

- New or improved Claude Code commands (`.claude/commands/`)
- Better voice profile examples or templates
- Documentation improvements
- Bug fixes in templates or instructions

## What to Avoid

- Adding real company content or personal information
- Changes that make the template less generic
- Adding dependencies (this is a markdown-only repo)

## Reporting Issues

Open an issue on GitHub.  Include:
- What you expected
- What happened
- Steps to reproduce (if applicable)
