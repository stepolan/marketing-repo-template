# Samples

Drop existing writing here for voice extraction.  Blog posts, LinkedIn posts, newsletters, conference talks, whatever represents this author's voice.

**Formats:** Markdown preferred.  Plain text works.  Copy-paste from LinkedIn or a blog is fine.

**Volume:** 10+ pieces is ideal.  More samples means a more accurate voice profile.

**Then run:** `/extract-voice caesar`

Claude will read everything in this directory and generate a voice profile at `authors/caesar/voice/voice-profile.md`.

**After extraction:** You can keep these samples for reference or delete them.  They are not used by any other command.
